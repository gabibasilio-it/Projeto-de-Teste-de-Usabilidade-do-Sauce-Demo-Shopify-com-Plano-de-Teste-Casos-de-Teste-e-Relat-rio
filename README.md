# Projeto de Teste de Usabilidade – Sauce Demo Shopify

## Objetivo

Avaliar a usabilidade das funcionalidades de Cadastro, Login e Recuperação de Senha do sistema Sauce Demo Shopify.

## Escopo dos Testes

* Cadastro de usuário
* Login
* Recuperação de senha

## Cenários Executados

1. Cadastro de usuário com dados válidos
2. Cadastro com e-mail já cadastrado
3. Cadastro sem preencher campos obrigatórios
4. Login com credenciais válidas
5. Login com senha inválida
6. Login com e-mail inválido
7. Solicitar recuperação de senha
8. Solicitar recuperação de senha com e-mail inválido
9. Solicitar recuperação de senha sem informar e-mail
10. Solicitar recuperação com e-mail não cadastrado

## Resultados

| Métrica                      | Resultado |
| ---------------------------- | --------- |
| Total de Cenários Executados | 10        |
| Cenários Aprovados           | 6         |
| Cenários Reprovados          | 4         |
| Taxa de Sucesso              | 60%       |

## Bugs Encontrados

* BUG-001 – Falha na validação do e-mail informado.
* BUG-002 – Ausência de mensagem clara para e-mail inválido.
* BUG-003 – Falta de feedback para e-mail não cadastrado.
* BUG-004 – Fluxo de recuperação de senha pouco intuitivo.
